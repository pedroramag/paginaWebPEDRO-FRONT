NOMBRE DEL PROYECTO: NOTWALL

ESTADO: FINALIZADO

DURACIÓN: Abril 2024 - Mayo 2024

GRUPO: Individual

ROL: FRONT, no se requiere back

TECNOLOGÍA: Java Android Studio

DESCRIPCIÓN: Proyecto final de FP. Se trata de una aplicación Android la cual está pensada para hacer cálculos de forma rápida relacionados con las inversiones. Por ejemplo: ¿Si compro en este precio y vendo en este cuánto habré ganado o perdido? ¿Y a cuánto porcentaje equivale ese número? Aparte de esos dos cálculos y alguno más también tenía un bloc de notas donde podías ir tomando apuntes, guardando notas, etc. Todo esto se guardaba dentro de la sesión del usuario y para hacer que el bloc de notas fuera seguro en caso de robo, una vez el usuario cerrase la aplicación el contenido desaparecía y la hoja se ponía en blanco. Para volver a verlo y poder trabajar con lo previamente hecho teníamos que darle a un botón de reseteo y este solicitaba una contraseña.
