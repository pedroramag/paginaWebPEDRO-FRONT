VERSIÓN 1.1
ADAPTACIÓN PARA MÓVILES, TABLETS Y PEQUEÑAS PANTALLAS
En esta versión el objetivo es adaptar la página web para que pueda utilizarse de forma cómoda y sencilla desde cualquier dispositivo. La interfaz ajustará automáticamente su diseño para ofrecer una correcta visualización tanto en teléfonos móviles y tablets como en ordenadores, mejorando así la experiencia de todos los usuarios. En resumen, la web pasará a ser responsive, adaptándose a cualquier dispositivo y tamaño de pantalla.

VERSIÓN 1.2
APLICACIONES "HIPOTECA" E "HIPOTECA INVERSIÓN INMUEBLES"
En está versión el objetivo es habilitar las aplicaciones que reciben el nombre de "CALCULADORA HIPOTECA" Y "CALCULADORA HIPOTECA INVERSIÓN INMBUBLES". Cada aplicacion esta diseñada para hacer preguntas específicas con las cuales se calcula de forma fácil, sencilla y automática el resultado final. De esta manera se conseguirá que la experiencia del usuario sea lo mas sencilla, cómoda y agradable posible a la par que entretenida.
