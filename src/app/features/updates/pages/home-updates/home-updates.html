ACTUALIZACIONES FINALIZADAS
VERSIÓN 1.1
ADAPTACIÓN PARA MÓVILES, TABLETS Y PEQUEÑAS PANTALLAS
En esta versión el objetivo es adaptar la página web para que pueda utilizarse de forma cómoda y sencilla desde cualquier dispositivo. La interfaz ajustará automáticamente su diseño para ofrecer una correcta visualización tanto en teléfonos móviles y tablets como en ordenadores, mejorando así la experiencia de todos los usuarios. En resumen, la web pasará a ser responsive, adaptándose a cualquier dispositivo y tamaño de pantalla.
FINALIZADA
ACTUALIZACIONES PENDIENTES
VERSIÓN X.X
INICIO DE SESIÓN EN EL APARTADO DE APLICACIONES
En esta versión será necesario iniciar sesión en el apartado de "APLICACIONES" para poder utilizar cualquiera de ellas. Esto permitirá que, en futuras actualizaciones, las aplicaciones puedan guardar y sincronizar datos cuando sea necesario o esté disponible dicha funcionalidad.

VARIAS VISTAS/ESCRITORIOS EN APLICACIONES
En está versión se crearán varios escritorios, el primero didicado a aplicaciones de gestión y el segundo dedicado a aplicaciones de inversión.
PRÓXIMAMENTE
VERSIÓN X.X
APLICACIONES "HIPOTECA" E "HIPOTECA INVERSIÓN INMUEBLES"
En está versión el objetivo es habilitar las aplicaciones que reciben el nombre de "CALCULADORA HIPOTECA" Y "CALCULADORA HIPOTECA INVERSIÓN INMBUBLES". Cada aplicacion esta diseñada para hacer preguntas específicas con las cuales se calcula de forma fácil, sencilla y automática el resultado final. De esta manera se conseguirá que la experiencia del usuario sea lo mas sencilla, cómoda y agradable posible a la par que entretenida.
PRÓXIMAMENTE
VERSIÓN X.X
APLICACIÓN DE GESTIÓN DE TAREAS
En está versión el objetivo es crear y habilitar una aplicación en la que podas anotar tareas, reuniones, apuntes, etc. Todo esto para que el usuario registrado pueda tener un orden, organización y gestión del tiempo.
PRÓXIMAMENTE
VERSIÓN X.X
APLICACIONES O ASISTENTE DE INVERSIÓN EN ACTIVOS BURSÁTILES
En está versión el objetivo es poder tener nuestro propio asistente o aplicacion el cual nos ayudará a crear un plan personalizado en función del riesgo que esté dispuesto a cometer cada usuario.
