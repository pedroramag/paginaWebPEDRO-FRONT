VERSIÓN 1.1
APLICACIONES "HIPOTECA" E "HIPOTECA INVERSIÓN INMUEBLES"
En está versión el objetivo es habilitar las aplicaciones que reciben el nombre de "CALCULADORA HIPOTECA" Y "CALCULADORA HIPOTECA INVERSIÓN INMBUBLES". Cada aplicacion esta diseñada para hacer preguntas específicas con las cuales se calcula de forma fácil, sencilla y automática el resultado final. De esta manera se conseguirá que la experiencia del usuario sea lo mas sencilla, cómoda y agradable posible a la par que entretenida.
