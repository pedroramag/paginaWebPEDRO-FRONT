SOBRE MI WEB

Esta web nace con el objetivo de compartir quién soy, cómo pienso y cómo evoluciona mi camino como desarrollador. Aquí encontrarás mis proyectos, conocimientos, experiencias y aplicaciones, pero también una parte más personal que irá creciendo con el tiempo para que cualquier persona que visite la web pueda conocerme mejor y seguir mi evolución tanto profesional como personal.

¿CÓMO SE HA DESARROLLADO LA WEB?
FRONTEND +
Toda la web ha sido desarrollada utilizando Angular y TypeScript. El objetivo ha sido construir una aplicación moderna, escalable y preparada para seguir creciendo con nuevas funcionalidades en el futuro.
HERRAMIENTAS UTILIZADAS +
Durante el desarrollo se han utilizado herramientas como GitHub para el control de versiones y Codex como apoyo en determinadas tareas de programación y desarrollo.
EVOLUCIÓN DEL PROYECTO +
La web está diseñada para evolucionar constantemente. Se incorporarán nuevas secciones, aplicaciones, mejoras visuales y funcionalidades a medida que avance mi formación, experiencia y conocimientos.
¿CÓMO FUNCIONA LA WEB?
QUIÉN SOY +
Una breve historia sobre mí, mi evolución personal y profesional. Con el tiempo esta sección irá creciendo para ofrecer una visión más cercana y completa de quién soy.
EXPERIENCIA LABORAL +
Esta sección recoge mi experiencia profesional y se actualizará a medida que participe en nuevos proyectos, empresas y retos laborales.
ESTUDIOS +
Formación académica, cursos, certificaciones y cualquier aprendizaje que forme parte de mi desarrollo profesional actual y futuro.
PROYECTOS +
Todos los proyectos personales, académicos y profesionales en los que he trabajado, trabajo actualmente o desarrollaré en el futuro.
APLICACIONES +
Aplicaciones funcionales desarrolladas por mí y disponibles para cualquier visitante. Actualmente incluyen herramientas de cálculo hipotecario y de inversión inmobiliaria, entre otras futuras aplicaciones.
ACTUALIZACIONES +
Historial de mejoras implementadas en la web, nuevas funcionalidades y futuras actualizaciones previstas para el proyecto.
